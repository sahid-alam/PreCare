# Design Features To Review

The `precare-design/` bundle includes several interface affordances that are not backed by current product or backend behavior. They were not implemented as working production features in this redesign pass.

## Admin Overview

- Pause/resume live stream control.
- Interactive table filters with persisted counts by tier.
- Fully data-driven 24-hour volume chart.

## Session Detail

- Export PDF from admin session detail.
- Flag session for clinical review.
- Open in ER queue.
- Replay audio.
- View protocol.
- Notify ER receiving doctor.

## Data/Infrastructure

- ER queue integration.
- Audio recording storage/replay.
- Clinical protocol versioning and display.
- Staff note composer and admin review persistence.
